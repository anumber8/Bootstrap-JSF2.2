Bootstrap-JavaEE7
=================

Convenient project skeleton for Java EE 7 Web application based on Twitter Bootstrap.

Technologies
------------

- Maven 3.0.5
- Glassfish 4.0
- Java EE 7
- JSF 2.2
- Omnifaces 1.5
- Twitter Bootstrap 2.3.1

Features
--------

- Facelets template Twitter Bootstrap
- Omnifaces 1.5 for automatic resources handling
